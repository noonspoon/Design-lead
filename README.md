# Jen Noon — Personal Site

A minimal personal website for Jen Noon, Product Design Lead at [frog](https://frog.co).

## Stack

- Plain HTML & CSS — no frameworks or build tools
- [Geist](https://fonts.google.com/specimen/Geist) via Google Fonts

## Structure

```
index.html   — single-page markup
styles.css   — all styles and responsive layout
```

## Running locally

Open `index.html` directly in a browser — no server required.
